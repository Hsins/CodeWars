# Get character from ASCII Value

## Information

- KYU: 8
- Link: [Get character from ASCII Value](https://www.codewars.com/kata/get-character-from-ascii-value/train/python)

## Description:

Write a function which takes a number and returns the corresponding ASCII char for that value.

Example:

```
get_char(65) # => 'A'
```

## My Solution

```python
def get_char(c):
  return chr(c)
```

## Lambda Solutions

```python
get_char = lambda c: chr(c)
```
